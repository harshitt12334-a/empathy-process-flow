## Project Overview
This project represents an Empathy Process Flow for a Smart Parking System designed to solve parking-related problems in smart citi


## Empathy Process Stages
1. User faces parking problem
2. User searches for smart solution
3. User uses Smart Parking App
4. User books parking slot
5. User reaches destination easily
6. User feels satisfied
